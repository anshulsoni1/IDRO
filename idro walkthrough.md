# IDRO Frontend Walkthrough

The **Intelligent Disaster Resource Optimizer (IDRO)** is a futuristic, AI-driven disaster management platform. It provides a real-time command center interface for coordinating relief efforts, monitoring disasters, and deploying resources.

## Interface Overview

The frontend features a sleek, dark-themed dashboard with neon accents, designed to feel like a high-tech disaster command room.

### 1. Command Center (Home)
The primary interface showing a live satellite map of India and surrounding regions. It provides quick access to major system modules and real-time alerts via the **Intel Stream**.

### 2. Active Disaster Feed
A dedicated monitoring screen for live disaster updates from satellite networks. It alerts operators to new events requiring intervention.

### 3. Impact Analysis & Mission Control
*   **Impact Analysis:** AI-generated models that calculate humanitarian needs and identify resource gaps.
